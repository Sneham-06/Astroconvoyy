## 🎯 Mission Brief
**AstraConvoy** is a high-precision military logistics platform. It leverages rule-based AI to optimize convoy routes, predict multi-factor threats, and manage mission-critical deployments with zero-latency emergency response.

---

## ⚡ Tactical Features

| Feature | Intelligence Provided |
| :--- | :--- |
| **⚠️ Threat Prediction** | 10-point scoring based on terrain, weather, and isolation. |
| **🎯 Route Optimization** | AI-selected primary and backup tactical routes. |
| **🚨 Auto-SOS System** | Instant emergency detection and rerouting protocols. |
| **⭐ Priority Engine** | Mission-based allocation (Medical, Ammunition, Supplies). |
| **📡 V2V Communications** | Secure vehicle-to-vehicle tactical intelligence link. |

## 🚀 Deployment Orders

### Prerequisites
- Node.js 18+ & Python 3.8+

### Quick Start
```bash
# 1. Initialize Intelligence Link (Backend)
cd backend && pip install -r requirements.txt
python app.py

# 2. Deploy Command Interface (Frontend)
npm install
npm run dev
```
*Access HQ at: `http://localhost:3000`*

## 📊 Command Center API
- `POST /api/convoy/create` — Initialize mission & threat analysis.
- `GET /api/convoy/list` — Real-time tracking of active fleet.
- `POST /api/driver/sos` — Immediate emergency protocol trigger.
- `GET /api/analytics/dashboard` — High-level strategic metrics.
